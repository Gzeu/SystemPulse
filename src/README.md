# Source Code

Contains the main application and supporting projects.

## Projects

### SystemPulse.App
WinUI 3 desktop application for Windows system monitoring.

**Folder Structure:**
- `Views/` - XAML pages and UI
- `ViewModels/` - MVVM ViewModels
- `Models/` - Data models
- `Services/` - Business logic services
- `Helpers/` - Utility functions
- `Resources/` - Styles, colors, typography
- `Properties/` - Project metadata

### SystemPulse.Core (Optional)
Shared business logic that could be used by other projects.

### SystemPulse.Tests
Unit and integration tests.
